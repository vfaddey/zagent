Ты опытный Python backend разработчик. Сгенерируй минимальное, но правильно структурированное FastAPI приложение с использованием SQLite.

Требования:

1. Стек:
- Python 3.11+
- FastAPI
- SQLAlchemy 2.0 (async)
- SQLite (через aiosqlite)
- Pydantic v2

2. Архитектура:
- Разделение на слои:
  - main.py (точка входа)
  - db.py (инициализация БД)
  - models.py (ORM модели)
  - schemas.py (Pydantic схемы)
  - crud.py (работа с БД)
  - routers/ (эндпоинты)

3. Функционал:
Реализовать CRUD для сущности "Item":
- id (int, primary key)
- name (string)
- description (string, nullable)

Эндпоинты:
- POST /items
- GET /items
- GET /items/{id}
- PUT /items/{id}
- DELETE /items/{id}

4. База данных:
- SQLite файл (app.db)
- Автоматическое создание таблиц при старте
- Асинхронная работа через AsyncSession

5. Дополнительно:
- Dependency injection для сессии БД
- Валидация входных данных
- HTTP коды ответов
- Пример curl запросов

6. Формат ответа:
- Сначала структура проекта (дерево файлов)
- Затем полный код каждого файла
- Без лишних объяснений
- Код должен быть готов к запуску

7. Запуск:
Добавь команды:
- установка зависимостей
- запуск через uvicorn

Не упрощай архитектуру до одного файла. Код должен быть близок к production-минимуму.
